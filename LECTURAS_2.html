HTML5 Conceptos Básicos


El diseño web o la programación de HTML se realiza inicialmente con el uso del block de notas. Que es HTML5? Es un estándar o un lenguaje de etiquetas que no llegan a reunir todos los requisitos que exige la tecnología de hoy. Es un código que esta entre los simbolos de mayor que o menor que, es así que una etiqueta tiene dos simbolos en la cual una se abre y la otra se cierra, hay etiquetas unas dentro de otras, Antes de la primera etiqueta hay que especificar en que versión vamos a trabajar para esto hay que insertar una etiqueta doctype, para lo cuual comienza con el simbolo mayor que, seguida del símbolo de admiración, luego la palabra DOCTYPE html y cierra con el símbolo memor que. Esta nomenclatura es nueva en HTML5 que quiere decir que es la nueva forma de programar en HTML5, es la primera etiqueta, todo siguiendo el orden establecido. Siempre todo terminara con una etiqueta de cierre. Todas las etiquetas deben de ser en minúscula. Para seguir realizando la programación HTML5 seguimos con la etiqueta HEAD es así que se abre símbolo menor que luego la palabra HEAD y cierra con simbolo mayor que. Que incluye el HEAD? Es una etiqueta que incluye la información importante de diverso tipo, como archivos javascript, iconos, titulos, etc. Dentro del HEAD van otras etiquetas como son meta charset, especifica como será presentada nuestra página web, meta name, especifica descripción y palabras claves, la meta name description, da una descripción clara importante para los buscadores, importante para caracterizar la págna web. La etiqueta meta name keywords, lleva un atributo como el lenguaje. La etiqueta BODY manejamos el cuerpo del archivo o el texto. Como se guarda el archivo de nuestra página web? Nos vamos al menu archivo, renombramos poniendo la extensioón . HTML5 y la podemos guardar en una carpeta en el escritorio. Si la vamos a abrir lo podemos hacer con el botón derecho.


ETIQUETAS, TITLE, LINK ESTRUCTURA DEL CUERPO EN HTML5


En la programación de HTML5 podemos revisar las siguientes etiquetas. La Etiqueta TITLE irá dentro de la etiqueta HEAD, tiene varios propósitos es colocar la información que sirve de ruta para un buscador, su labor fundamental es utilizar la palabra clave para que se posicione en los buscadores. Se coloca la etiqueta de entrada, simbolo menor que, la palabra title, cierra con simbolo mayor que, luego coloca la palabra clave para los buscadores y luego hace el cierre de etiqueta con los simbolos menor que y mayor que y la palabra title.

La etiqueta LINK, va dentro del HEAD, puede ir despues de la etiqueta TITUE, La etiqueta LINK sirve para informar de que tipo de archivo estamos agregando, sirve para enlazar o incluir documentos adicionales a nuestra pagina web como hojas de estilo o documentos javascript, como tambien informa la ruta en la que se va a incluir.


ETIQUETA BODY, es el cuerpo del archivo, inicialmente se utilizaban dos etiquetas como son una etiqueta table, que utilizaba 3 filas y tres columnas, tambien se utlizó la etiqueta div, servía para hacer divisiones y crear una división para insertar una imagen, otra división para crear una barra de navegación. En la actualidad estas etiquetas ya no son utlizadas y el HTML5 utiliza nuevas etiquetas que utilizan el sitio mas organizado, la representación gráfica puede incluir una cabecera donde incluye un logo o información de la empresa, tiene una parte donde aparece la barra de navegación donde aparecen menus, pestañas, menús desplegables para navegar que sería lo que es la pagína web . Suelen tener un contenido principal donde esta la información importante, una barra lateral donde aparece los tipicos enlaces para navegar dentro del sitio web, la cual puede estar a la derecha o a la izquierda. Suelen tener una zona inferior donde aparece información del dueño de la pagina, como dirección, telefono, email. Para crear un titulo podemos incluir un header, uno que se abre y otro que se cierra. pero dentro del header se abre un h1, con etiqueta que se abre y otra que se cierra, que sirve para resaltar un texto.
